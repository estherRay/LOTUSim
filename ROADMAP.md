The LOTUSim roadmap is published biannually and highlights priority topics and features for upcoming releases. See [Contributing](contributing) if you'd like your work to align with these priorities.

## Released

### November 2025 — First Public Release
The first public release of LOTUSim.

## In Progress

### First Half of 2026

New capabilities coming in the next biannual release:

- **Synthetic Radar** — Integrated synthetic radar development
- **Example Scenarios** — Pre-made scenarios so users can get started with LOTUSim quickly
- **Scenario Creation via Web UI** — Build scenarios through a web interface for a faster workflow
- **Power Manager** — Simulates batteries and generators, tracks sensor and thruster energy use, and applies custom emergency logic when energy runs low (e.g., deactivating low-priority sensors, reducing speed)

### July 2026 — Public Release
New version including all the capabilities above. The first conference of LOTUSim happened in July 2026 to talk about the new features and show some concrete use cases of our users.

## Planned

### Second Half of 2026

- **Unreal Engine integration** — in addition to existing Unity support
- **Modular sensor plugin interface** — for custom sensors
- **Addition of new models**
